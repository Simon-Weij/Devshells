# Devshells

A collection of devshells with nix for various languages/frameworks

Currently has devshells for:

- Flutter
- Golang
- Wails
- Pyside
- Python
- Rust
- Tauri
- Bun
- Electronbun

## Contributing

Do you have a devshell you want here? Feel free to open a PR
